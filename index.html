Diseño Web

El diseño web es un área enfocada en el desarrollo de interfaces digitales, como el diseño de sitios y aplicaciones para web. Para ello, los diseñadores web crean las páginas utilizando código HTML, CSS y Java Script. La clase de Diseño web es donde se aprende a diseñar y programar páginas web. El código se edita en un editor de texto para compilarlo en Wamp, o en cualquier navegador

Programacion

En la clase de programación realizamos un proceso por medio del cual se diseña y se codifica. Por medio de lenguajes de programación, ya sea en Pseint, C++, C#, PHP, Phtyton, Java Script, entre otros. De acuerdo con ellos el código se escribe, se prueba y se perfecciona

Mantenimiento y reparación de Computadoras

El mantenimiento es la revisión de equipos en funcionamiento para garantizar su buen desempeño, tanto de hardware como de software en un ordenador o PC. En esta asignatura nos encargamos del mantenimiento correctivo de computadoras de la reparación y sustitución de los componentes de hardware y software que se averían. Tambien sea para corregir fallas existentes o para prevenirlas.

Laboratorio

El laboratorio de informática es una clase muy completa para el entrenamiento de la comprensión de programas de informática, tales como Excel, Word, Photoshop, Illustrator, AutoCAD, HTML5, etc. Son espacios para estudiar, experimentar y aprender el funcionamiento de programas de informática y practicar su uso. Gracias a la clase del laboratorio de informática, los profesores tienen un mayor control de sus alumnos y pueden reforzar aquellas actividades que los alumnos no terminan de asimilar

Redes Infórmaticas

La asignatura de redes infórmaticas plantea como un recorrido por las distintas tecnologías de interconexión de redes, tanto en arquitecturas locales como de área extensa. Este recorrido se refleja en un estudio pormenorizado del funcionamiento de los principales protocolos de red(desde el nivel físico hasta el de aplicación). También se analizan las tecnologías de última generación en conexiones inalámbricas.
